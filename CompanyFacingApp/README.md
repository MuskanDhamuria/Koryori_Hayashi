
# CompanyFacingApp

Staff dashboard for analytics, inventory alerts, pricing insights, and AI assistance.

## Run

```powershell
cd CompanyFacingApp
npm install
npm run dev
```

## Backend Dependency

This app expects the backend to be running, by default at `http://localhost:4000`.

Optional env:

```env
VITE_API_BASE_URL=http://localhost:4000
```

## Current Scope

- Staff login with backend JWT
- Revenue and operations analytics
- Inventory alert views
- Pricing and comparison views
- AI assistant panel
